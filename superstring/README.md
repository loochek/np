# Задача о суперстроке

Формат input.txt:
Строки.
Пример:
```
абвгд
ааа
абракадабра
...
```

Цель: найти как можно более короткую строку, такую, чтобы все строки из input.txt были ее подстроками.

Формат output.txt:
Итоговая суперстрока.

Первая цель: пробить 26000, т.е. найти суперстроку длины < 26000.

## Решение

Для решения задачи был реализован следующий алгоритм - рассматривается полный граф на множестве строк, где весом ребра является длина перекрытия, затем жадным алгоритмом все строки объединяются в цепочки по перекрытиям. Также на этом графе были применены две локальных оптимизации  - 2-opt (аналогично распутыванию петель в TSP) и перестановки произвольных строк

Полученный результат для предоставленных входных данных - суперстрока длины 17431

Данный алгоритм фактически является жадным алгоритмом из конспекта, следовательно, для него применимы те же соображения о коэффициенте аппроксимации (гипотеза - алгоритм является 2-приближенным)
