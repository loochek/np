# Метрическая задача коммивояжера

Цель: найти как можно более короткий гамильтонов цикл

Формат input.txt:
[номер вершины] [координата x] [координата y]
Пример:
```
1 -1583.3869220805623 -5976.506597083639
2 2543.2042981492314 606.2923404220139
3 -1039.9276416868006 -466.54079359682146
4 -1006.5614628103558 -4412.748876528386
5 -1318.3615977782024 -794.8025520995312
6 3230.9559189650054 2772.8463726627783
7 -787.2595945361504 -4494.9068984826845
8 -2231.2233820375714 -6328.5922607247385
9 -2963.668984141099 -3875.5585393484257
...
```


Стоимость ребра: евклидово расстояние

Формат output.txt:
Перечень вершин, составляющий гамильтонов цикл

Первая цель: пробить 1 млн, т.е. найти цикл длины меньше 1000000.

## Решение

Для решения задачи был реализован алгоритм Кристофидеса-Сердюкова из конспекта. Для улучшения ответа была применена локальная оптимизация по распутыванию петлей. Из конспекта известно, что такой алгоритм является 3/2-приближенным, при этом оценка является точной

Полученный результат для предоставленных входных данных - гамильтонов цикл длины 924819
